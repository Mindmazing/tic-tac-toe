# tic-tac-toe

2 player game directly based in tic tac toe game.

## Objectives

- No use of global variables
- Use concepts like:
- - Factory Functions
- - Enclosing
- - IIFE (Inmediatly Invoked Fucntion Expressions)

## Planning

### First Phase (Game code organization)

1. Start creating game logic in JavaScript
2. Create IIFE for game logic
3. Create Factory Functions for Players
